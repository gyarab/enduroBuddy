# Registration Flow Redesign — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add terms acceptance to signup, add Google first-time onboarding at `/accounts/profile-setup/`, and redesign email templates to dark theme with light-mode fallback.

**Architecture:** Backend adds `terms_accepted_at` to Profile, `needs_profile_setup` property, a new `/api/v1/auth/profile-setup/` endpoint, and exposes the flag in `auth_me`. Frontend adds a terms checkbox to signup, a Nuxt middleware guard, and a new `/accounts/profile-setup/` page in the auth shell. Email templates are rewritten with dark inline styles + `@media (prefers-color-scheme: light)` overrides.

**Tech Stack:** Django 5.2, allauth, Nuxt 3, TypeScript, Vitest, `@vue/test-utils`

        if self._should_redirect(request):
            return redirect(self.SETUP_PATH)
        return self.get_response(request)

- [ ] **Step 5: Add `auth_profile_setup` view at the end of `backend/api/views/auth.py`**

```python
@login_required
@require_http_methods(["POST"])
def auth_profile_setup(request):
    from django.utils import timezone
    user = request.user
    profile = getattr(user, "profile", None)

    if not profile or not profile.needs_profile_setup:
        return JsonResponse({"ok": False, "message": "Profile setup not required."}, status=400)

    payload = _json_body(request)

    role = payload.get("role", "")
    if role not in Role.values:
        return JsonResponse(
            {"ok": False, "errors": {"role": ["Zvolte roli sportovce nebo trenéra."]}},
            status=400,
        )

    if not payload.get("terms_accepted"):
        return JsonResponse(
            {"ok": False, "errors": {"terms_accepted": ["Souhlas s podmínkami použití je povinný."]}},
            status=400,
        )

    first_name = payload.get("first_name", "").strip()
    last_name = payload.get("last_name", "").strip()
    if first_name:
        user.first_name = first_name
    if last_name:
        user.last_name = last_name
    user.save(update_fields=["first_name", "last_name"])

    profile.terms_accepted_at = timezone.now()
    profile.google_role_confirmed = True
    profile.google_profile_completed = True
    profile.save(update_fields=["terms_accepted_at", "google_role_confirmed", "google_profile_completed"])
    profile.apply_role(role)

    redirect_to = _app_url("/coach/plans" if role == Role.COACH else "/app/dashboard")
    return JsonResponse({"ok": True, "redirect_to": redirect_to})
```
